# Exploratory Data Analysis (EDA) Project
# Airbnb Market EDA

An advanced Exploratory Data Analysis project on New York Airbnb listings.

---

## Project Overview

This project analyzes Airbnb market trends using real-world listing data from New York City.

The goal is to uncover:

- Pricing trends
- Neighborhood performance
- Customer behavior
- Listing availability

---

## Dataset

Source:

NYC Airbnb Open Data from Kaggle

Contains:

- 48,000+ records
- 16 features

Key Features:

- Price
- Room Type
- Neighborhood
- Reviews
- Availability

---

## Tools Used

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

---

## EDA Workflow

### Data Cleaning

- Missing value treatment
- Duplicate removal
- Outlier handling

### Statistical Analysis

- Descriptive statistics
- Correlation analysis

### Visualization

- Histograms
- Boxplots
- Countplots
- Heatmaps

---

## Key Findings

- Manhattan has the highest prices.
- Entire homes generate premium pricing.
- Reviews and availability affect occupancy.
